v5.1 (25.02.2016)

Созданы живые обои показывающие карту на текущий момент. Добавлен расчет мунданного положения точек (в плоскости системы домов). Части углового аспекта теперь можно задавать абсолютно (в градусах). Теперь можно исключать части углового аспекта для отдельных точек или категорий точек. В набор классических угловых аспектов добавлены поражения Солнцем (казими, сожжение, в лучах). В API добавлен поиск точки карты по ее координатам на экране. Во всех встроенных рабочих столах при нажатии на символ точки переключается ее аспектация. Исправлены ошибки.

v5.0 (7.10.2015)

Замена Lua на язык BeanShell (Лучше скорость и использование памяти. Полный доступ к Android API). Добавлена возможность контролировать пересечение границы знака в угловых аспектах. Во всех рабочих столах можно отключать аспекты которые пересекают границу знака. Добавлена возможность переопределять названия знаков Зодиака (локализация, замена на символы итд) Информация о положениях точки может показывать символ Зодиака (через Меню->Настройки). Исправлена ошибка сброса системы домов в окне редактирования карты. Исправлена ошибка в расчетах дирекций. Создана версия для стационарного использования AstroMantaPC

Отрисовка изображений Lua скриптами внутри области карт. Вставка в область информации изображений отрисованных Lua скриптами (графики, диаграммы, итд). Добавлен новый тип аспектов по знаку (без учета орбиса). В рабочий стол хораров добавлены аспекты по знаку.

Информация о положении точек карты выводится для всех видимых точек. Реализована многоязычность. Все рабочие столы поддерживают английский и русский языки.

В рабочие столы синастрий добавлены расчеты предыдущих новолуний в наталах. Добавлен день недели в информацию момента карты. Исправлена ошибка вывода информации рабочего стола "Sinastry multy".
